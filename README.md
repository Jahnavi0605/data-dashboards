## 💡 My Dashboard Design Principles
1. **Start with the business question** — not the data
2. **Choose the right chart type** — bar for comparison, line for trends, map for geography
3. **Minimise cognitive load** — one key insight per view
4. **Design for the audience** — executive summary vs operational detail
